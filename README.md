## 🚀 Features
- 📖 Add, edit, and delete books
- 🔍 Search books by title or author
- 📊 Track reading progress with statuses (To Read, Reading, Finished)
- 🎨 Beautiful and minimal UI

## 🛠 Tech Stack
- **Language**: Java ☕
- **Database**: SQLite 🗄️
- **UI**: XML + Material Design 🎨
